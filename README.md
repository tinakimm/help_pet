## 개요
프로젝트: 2023학년도 1학기 캡스톤 프로젝트 "구해줘 펫즈"

기획 및 제작: 김수인, 신민경(https://github.com/sinmingyeong), 황민정(https://github.com/hmjas)

제작 기간: 2023.4~2023.6

주요 기능: 동물병원 위치 정보, 길찾기 서비스 제공, 평점에 따른 마커색 구분
